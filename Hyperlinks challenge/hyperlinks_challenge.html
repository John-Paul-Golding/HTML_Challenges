The Journey so Far

I started this course on the 28 February 2018 not knowing exactly what to expect but I knew one thing and that was that it wasn't going to be all sunshine and rainbows. So far my coding journey has been rollercoaster ride for me. Sometimes I understand what I have to do and other times I feel as if I am drowning. But as I always say, the most fun part of the rollercoaster is the way down! In this module, HTML, I have learned how to write an HTML document, add headings (h1,h2,h3,h4,h5 and h6), write paragraphs, add images from folders and links, add links, forms and tables, as well as all the HTML tags that come with each element. With these skills I am able to produce a skeleton webpage.
